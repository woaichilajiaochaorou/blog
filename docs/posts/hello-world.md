# Hello World

这是博客的第一篇文章，欢迎来到我的个人技术博客！

## 关于这个博客

这个博客使用 [VitePress](https://vitepress.dev/) 构建，部署在 Oracle Cloud ARM 实例上。

技术栈：

- **静态生成**：VitePress
- **容器化**：Docker + Nginx
- **服务器**：Oracle Cloud ARM（4 OCPU / 24GB）
- **CI/CD**：GitHub Actions 自动部署
- **HTTPS**：Let's Encrypt + Certbot

## 为什么选择这套方案

1. **VitePress** - 基于 Vite，构建速度快，Markdown 体验一流
2. **Oracle Cloud ARM** - 永久免费层，4 核 24G 性能远超 GitHub Pages
3. **Docker** - 环境隔离，一键部署，迁移方便
4. **GitHub Actions** - push 代码自动部署，零人工干预
